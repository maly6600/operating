Simulates unexpected process failure.

Authors: Karl Marklund (karl.marklund@it.uu.se).

Description

Simulates unexpected process failure. Erlang can be used to construct very robust systems that can handle and recover from unexpected errors. In order to test some of these features we need a way to introduce random errors.
